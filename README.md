# Burpy - Burp Suite http session viewer

## Getting Started

First, run the development server:

```bash
npm run dev
```

```bash
yarn dev
```

```bash
pnpm dev
```

Open [http://localhost:3069](http://localhost:3069) with your browser to see the result.
